# Portfa

![Portfa](http://f.cl.ly/items/2X1A1u0J263G150u0c1R/Portfa.png)

Create a basic portfolio with just a few clicks.

## Run
```shell
# You need memcached running for session storage
$ git clone
$ bundle install
$ rake db:setup
$ foreman start -f Procfile.dev
```
